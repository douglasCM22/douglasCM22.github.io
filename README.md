# Interactive Resume

This is a modern, interactive, and responsive resume built with React, TypeScript, and Vite. It's designed to be a dynamic and visually appealing showcase of my professional experience, skills, and projects.

The project is deployed on GitHub Pages and can be accessed at [https://douglascm22.github.io/](https://douglascm22.github.io/).

## Features

- **Interactive Timeline:** A chronological timeline of my professional experience, with detailed descriptions of each role.
- **Skills Analytics:** A data-driven visualization of my technical skills, using charts to represent my proficiency in different areas.
- **Impact Highlights:** A section dedicated to showcasing key achievements and their impact.
- **Responsive Design:** The layout is optimized for all screen sizes, from mobile devices to desktops.

## Tech Stack

- **React:** A JavaScript library for building user interfaces.
- **TypeScript:** A typed superset of JavaScript that compiles to plain JavaScript.
- **Vite:** A fast and lightweight build tool for modern web projects.
- **Tailwind CSS:** A utility-first CSS framework for rapid UI development.
- **Chart.js:** A flexible and customizable charting library.
- **Framer Motion:** A library for creating production-ready animations.

## Getting Started

To run the project locally, follow these steps:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/douglascm22/douglascm22.github.io.git
    ```

2.  **Install dependencies:**

    ```bash
    cd douglascm22.github.io
    npm install
    ```

3.  **Run the development server:**

    ```bash
    npm run dev
    ```

    This will start the development server at `http://localhost:5173`.

4.  **Build for production:**

    ```bash
    npm run build
    ```

    This will generate a `dist` folder with the static files for production.
